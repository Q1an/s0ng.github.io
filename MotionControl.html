Hardware design


The Controller

This demo system is based on an open-source Arduino board with a wireless shield. The Arduino board is specially designed for serve control propose.

The Rotation of Robot Arm

Due to the limited torque provided by the simple serve motor, inspired by the Chinese round table, I designed a similar structure to guarantee its safety and performance.

4-DOF Test Robotic Arm

This 4-DOF robitic arm is a prototype to verfiy the stability and accuracy of the whole sytem.

Software design


Control Interface

This is the local version of the software. It provides all the raw data and an intuitive view of skeleton in 3D dimension. It also offers a platform for the operator to calibrate the system. In this software, operator can get access to the latest information from the robot and use body to control it to deal with all the situations.

Remote Software

This is the screen of remote laptop. This program is responsible to transfer the remote depth and video data, robot status and robotic arm position to the local computer. It also uses the local data to drive the robot.
